# MuseChat
An app where you can see and chat with people listening to the same song as you in real time. 
Utilizing spotify's [API](https://developer.spotify.com/documentation/web-api) and Apple Music [API](https://developer.apple.com/documentation)
Main Features:
  Apple Music and Spotify connectivity
  Live limited Chat(while music plays)
  Current Active Listeners Display
  User Map to see what others are actively listening to.
